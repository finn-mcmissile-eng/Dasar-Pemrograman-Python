## Materi Pembelajaran
Topik-topik yang telah saya pelajari berdasarkan modul perkuliahan:

### 1. Dasar-Dasar Python
- **Pengenalan Python:** Sejarah, instalasi, dan penggunaan interpreter Python.
- **Elemen Dasar:** Mengenal variabel, tipe data (integer, float, string), dan operator aritmatika.
- **Input & Output:** Cara mengambil input dari pengguna dan menampilkan hasil (*formatting*).

### 2. Struktur Kontrol & Logika
- **Struktur Percabangan:** Implementasi 'if', 'if-else', dan 'if-elif-else' untuk pengambilan keputusan.
- **Struktur Perulangan:** Menggunakan 'for' dan 'while' untuk eksekusi kode berulang.

### 3. Struktur Data & Koleksi
- **Tipe Data Koleksi:** Penggunaan List, Tuple, Dictionary, dan Set dalam Python.
- **String Processing:** Teknik manipulasi dan operasi pada string.

### 4. Konsep Lanjutan (Algoritma)
- **Stack (Tumpukan):** Konsep LIFO (*Last In First Out*).
- **Queue (Antrian):** Konsep FIFO (*First In First Out*).
- **Linked List & Tree:** Dasar-dasar struktur data hierarkis dan terhubung.
